# 📍 MapRuteGIS — Navigasi & Fasilitas Terdekat

**MapRuteGIS** adalah aplikasi WebGIS modern yang membantu pengguna menemukan rute terbaik sekaligus mengeksplorasi fasilitas penting di sekitar tujuan.

## 🚀 Fitur Unggulan
- 🛣️ **Rute Optimal:** Jalur tercepat menggunakan algoritma OSRM.
- 🚦 **Analisis Kemacetan:** Estimasi waktu perjalanan berdasarkan simulasi jam sibuk.
- 🏦 **Fasilitas Otomatis:** Cari Bank, SPBU, dan tempat makan di sekitar tujuan tanpa input manual.
- 🛰️ **Mode Satelit:** Beralih ke tampilan satelit untuk detail lokasi yang lebih jelas.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (ES6)
- **Maps:** [Leaflet.js](https://leafletjs.com/)
- **Routing:** Leaflet Routing Machine & OSRM
- **Data:** OpenStreetMap & Overpass API

## 📖 Cara Penggunaan
1. Masukkan lokasi asal atau gunakan tombol **📍 GPS**.
2. Masukkan lokasi tujuan (atau gunakan **Ctrl+Klik** pada peta).
3. Klik **Cari Rute**.
4. Lihat panel samping untuk informasi jarak, waktu tempuh, dan daftar fasilitas terdekat.
